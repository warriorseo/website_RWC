## 2. กฎการกรองข้อมูล (GA4 API & Segment Filters)
เมื่อดึงข้อมูล หรือสร้างลิงก์สำหรับ GA4 **ต้องระบุเงื่อนไขต่อไปนี้เสมอ** เพื่อให้ข้อมูลเป็น SEO Traffic จริงๆ:
- **Channel / Medium:** ต้องกรองเฉพาะ **Organic Traffic** เท่านั้น โดยเช็คจาก `sessionDefaultChannelGroup` ให้ครอบคลุม:
  - `Organic Search`
  - `Organic Video`
  - `Organic Social`
  - `Organic Shopping`

## 3. ตรรกะการสร้าง URL ลิงก์ตรงไปยัง GA4 (คอลัมน์ K และ P)
การสร้างลิงก์สำหรับเปิดใน Google Sheets จำเป็นต้องระมัดระวังเรื่อง **URL Encoding** สูงมาก เนื่องจาก Google Sheets จะแกะรหัส (Decode) `%` ออกไป 1 ชั้นเมื่อมีการคลิกผ่านชีต

### กฎการแปลงรหัส (Double-Encoding Rule)
เมื่อต้องการส่งเครื่องหมาย `/` ไปยังเบราว์เซอร์ปลายทางผ่านลิงก์ Google Sheets:
1. ค่าปกติ: `/thread-lift-vs-botox/`
2. เข้ารหัสชั้นที่ 1 (Single): `%2Fthread-lift-vs-botox%2F`
3. เข้ารหัสชั้นที่ 2 (Double): `%252Fthread-lift-vs-botox%252F`
*สคริปต์ Python ต้องสร้างค่าเป็นรูปแบบ Double-encoded เสมอ เพื่อป้องกันไม่ให้ลิงก์ปลายทางแสดงผลเป็นหน้าเปล่า (No Data)*

### พารามิเตอร์บังคับใน URL
เพื่อให้ฟิลเตอร์ Organic ทำงานทันทีเมื่อกดเปิดลิงก์:
- ต้องระบุ `_u..built_comparisons_enabled=true` ในพารามิเตอร์ `params` ของ GA4 เพื่อบังคับให้เปรียบเทียบฟิลเตอร์ทำงาน
- ต้องต่อท้าย URL ด้วย `&ruid=0d646679-6477-4cd0-a2cc-a3c33313c77f&collectionId=10576530072&r=15246722299` เสมอ เพื่อล็อค Session สถานะการทำงานของ Workspace ลูกค้า
